Людям

Этот сайт — мой первый полноценный проект. Цель это проекта — показать, что я умею, какими технологиями владею. Его можно разделить на три основные части:

1). Первая часть - серверный код. В нём реализована функциональность регистрации и входа в систему через JWT. Сайт использует публичный API и позволяет искать новости за прошедшую неделю и сохранять понравившиеся в личном кабинете. Избранные новости хранятся в базе данных Mongo DB на сервере. Этот сервер является отказоустойчивым и безопасным.

2). Вёрстка сайта выполнена по методологии БЭМ. Это делает сайт гибким и масштабируемым. Кроме того, добавлять новые страницы и функциональность проще и быстрее, чем в проектах, выполненных по другим методологиям. Также сайт проще, дешевле поддерживать и он адаптивен под любые устройства.

3). И естественно JavaScript, что оживить его и реализовать взаимодействие с публичными API и API собственного сервера. При создании проекта была использована сборка технологией Webpack. Это современный подход к разработке сайтов. Код оптимизируется, а значит, сайт работает быстрее. Также сайт работает в старых браузерах. Даже если посетители — пользователи старых операционных систем, у них не будет проблем с отображением. А значит, потенциальная аудитория возрастает.
